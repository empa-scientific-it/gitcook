## How to Contribute

This repository is a **template** designed for learning Git. Here's how to get started:

1. **Fork this repository** to create your own copy
2. **Clone your fork** to your local machine
3. **Add your recipe** following the template structure
4. **Commit your changes** with a clear message
5. **Push to your fork** and create a Pull Request

Each recipe contribution teaches you core Git skills: branching, committing, pushing, and creating pull requests.

## What You'll Learn

- Forking and cloning repositories
- Creating and switching branches
- Making commits with meaningful messages
- Pushing changes to remote repositories
- Creating pull requests
- Collaborating with others through code review
